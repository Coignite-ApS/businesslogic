# Tasks Backlog

Manage the platform task backlog. See `.claude/skills/tasks/SKILL.md` for full instructions.

## Usage

- `/tasks` — show full backlog status across all services
- `/tasks <service>` — filter by service (cms, ai-api, formula-api, flow, gateway)
- `/tasks pick-next` — recommend next task to work on
- `/tasks start <service>/NN` — mark task as in-progress and begin
- `/tasks complete <service>/NN` — mark as completed

## Execution

Read `.claude/skills/tasks/SKILL.md` and follow the instructions.
