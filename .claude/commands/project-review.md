# Project Review

Run a comprehensive project health check. See `.claude/skills/project-review/SKILL.md` for full instructions.

## Usage

- `/project-review` — full review (tests, git, services, database snapshot, docs, migration progress, next steps)
- `/project-review quick` — skip database dump and legacy comparison
- `/project-review report` — generate a markdown report file
- `/project-review plan` — review + re-evaluate migration plan

## Steps

1. Read `.claude/skills/project-review/SKILL.md` for the complete procedure
2. Execute all steps in order
3. Present findings as a summary table
4. Suggest next actions based on findings
